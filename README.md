eleme
====

饿了么网页
这是个仿饿了么的点餐页面，简单的三个主页面：点餐、评价、商家介绍。

页面多处用到 better-scroll 这个插件，实现：滑动浏览页面，切换菜单自动滚动到相应内容，滑动浏览图片；

技术栈
====
vue2 + + vue-router + webpack + ES6 + sass + better-scroll
